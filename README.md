# portfolio
# 🌊 Vidhi's Portfolio

> A creative underwater-themed developer portfolio built with React + Vite.

---

## ✨ Features

- 🐠 **Animated underwater ocean background** — interactive whale + video scenes
- 🃏 **Flip-card project carousel** — hover to reveal project details
- 🏆 **Experiences & Achievements page** — hackathons and awards with tabbed view
- 🌊 **Smooth section transitions** — gradient blends between all sections
- 📱 **Responsive layout**
- 🦶 **Glassmorphism footer**

---

## 🛠️ Tech Stack

| Category | Technology |
|----------|-----------|
| Framework | React 18 |
| Build Tool | Vite |
| Styling | Inline CSS + CSS-in-JS |
| Animation | CSS Keyframes |
| Language | JavaScript (JSX) |
| Deployment | GitHub Pages / Vercel |

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/vidhiwhere/portfolio.git

# Navigate into the project
cd portfolio

# Install dependencies
npm install

# Start the dev server
npm run dev
```
